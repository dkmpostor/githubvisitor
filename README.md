# GB VISITOR GITHUB

# How to use :

1. git clone https://github.com/dkmpostor/gbvisitor
2. cd gbvisitor
3. npm install
4. node index.js
5. Input your link
6. Done

# How to get link :
1. Open your github
2. Klik on your visitor picture on README.md
3. You will be redirect to newtab
4. Copy url
5. Use that url and input into the bot
